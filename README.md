# Startavive Cloud

Startavive Cloud is a premium web and application development platform designed to level up your digital presence.

## Features

- **Custom Development**: High-performance websites and custom applications tailored to your business needs.
- **Fast Delivery**: Accelerated project timelines without compromising quality.
- **Performance Optimized**: Sub-second load times and flawless user experiences across all devices.
- **Enterprise-Grade Security**: Advanced encryption, comprehensive safeguards, and strict access protocols.
- **SEO Optimized**: Built for search engines from day one to ensure high organic visibility.

## Setup

1. Clone this repository
2. Run `npm install` to install dependencies
3. Create a `.env` file with your configuration
4. Run `npm start` or `node server.js` to start the backend server
5. Open `http://localhost:5000` to view the website

## Tech Stack

- **Frontend**: HTML5, Tailwind CSS, Vanilla JS
- **Backend**: Node.js, Express.js
- **Database**: Google Sheets (via Google Spreadsheet API)
